1️⃣ JavaScript Array Methods Cheat Sheet

¡Bienvenido! Este repositorio es una guía completa diseñada para dominar el manejo de **Arrays** en JavaScript. 
Aquí encontrarás los métodos más utilizados, explicados breve con ejemplos de código y sus resultados.

📌 ¿Qué encontrarás aquí?

Cada método está documentado siguiendo esta estructura:
1.  Descripción: ¿Para qué sirve?
2.  Código: Ejemplo práctico y limpio.
3.  Resultado: Comentario con la salida esperada para facilitar el aprendizaje visual.

🛠️ Categorías de Métodos

Transformación y Filtrado
* map(): Crea un nuevo array transformando cada elemento.
* filter(): Filtra elementos basados en una condición.
* reduce(): Reduce el array a un único valor (acumulador).

📍 Búsqueda
* find(): Devuelve el primer elemento que cumple la condición.
* findIndex(): Devuelve el índice del primer elemento encontrado.
* includes(): Verifica si un elemento existe en el array.

✂️ Manipulación
* push() / pop(): Añadir o quitar al final.
* shift() / unshift(): Añadir o quitar al inicio.
* splice() / slice(): Cortar o modificar secciones del array.
